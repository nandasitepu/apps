Aplikasi Kawal Desa

Terakhir diperbarui: 22 April 2026

Kawal Desa berkomitmen untuk melindungi privasi warga dan pengguna. Kebijakan ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan menjaga data Anda dalam rangka transparansi publik.

1. Data yang Dikumpulkan

  • Data Identitas: NIK (hanya untuk verifikasi), Nama, dan Wilayah Desa.
  • Data Interaksi: Laporan pengaduan, partisipasi UMKM, dan riwayat akses laporan desa.
  • Data Perangkat: Alamat IP dan informasi teknis perangkat demi keamanan sistem.

2. Penggunaan Data

Data yang dikumpulkan digunakan semata-mata untuk:

  • Memverifikasi identitas pelapor dalam fitur pengaduan warga.
  • Menyediakan data statistik pembangunan desa bagi aparat dan publik.
  • Keperluan pengawasan intern pemerintah (APIP) dalam mencegah penyalahgunaan dana desa.

3. Penghapusan Data

Kami menghormati hak Anda atas data pribadi. Anda dapat meminta penghapusan akun dan data terkait kapan saja dengan ketentuan sebagai berikut:

  • Proses Mandiri: Melalui menu "Pengaturan Profil" > "Hapus Akun" di dalam aplikasi.
  • Permintaan Manual: Mengirimkan email ke admin@mydata.id dengan subjek "Penghapusan Data - Kawal Desa".
  • Waktu Proses: Data akan dihapus secara permanen dari server aktif kami dalam waktu 1x24 jam setelah verifikasi.

Catatan: Data laporan keuangan atau pengaduan yang sudah masuk dalam tahap audit APIP tidak dapat dihapus karena merupakan bagian dari catatan resmi negara.

4. Keamanan

Semua transmisi data dienkripsi menggunakan protokol keamanan standar industri. Data Anda disimpan di server terpercaya dengan perlindungan akses berlapis.
